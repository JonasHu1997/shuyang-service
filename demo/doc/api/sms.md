smsService API
-----------
[短信接口]
-----------

#### 接口功能
> 发送短信
#### 接口地址
> /sms
#### 返回格式
> JSON
#### 请求方式
> POST
#### 请求Content-Type
> JSON
#### 请求参数
| 参数 | 必选 | 类型   | 默认值 | 说明                                    |
|------|------|--------|--------|-----------------------------------------|
| phone ||string|| 手机号|
|phones||array| | 手机号数组，群发时传此参数|
|content||string||内容，不使用模板时直接发送内容|
|tpl_id||string||模板id，与内容2选1|
|tpl_data||object|| 模板数据，json子对象

> 地址：接口地址示例

   data: "",
   message: "发送成功"
}
